# 🎮 SET – Solo Time Attack (Unreal Engine 5)

![SET Game Thumbnail](/ScreenShot/thumbnail.png)

一人用タイムアタックゲーム「SET」を Unreal Engine 5 で制作しました。  
場にあるカードから SET（正しい組み合わせ3枚）を見つけ続け、組み合わせがなくなるまでの時間を競います。

---

## 📌 概要

- **制作期間:** 約1ヶ月  
- **担当:** 個人制作  
- **対象プラットフォーム:** Windows（Blueprint + UE5）  
- **技術スタック:** Unreal Engine 5, Blueprint, Python（カード生成用スクリプト）

---

## 🎯 開発の目的

- UE5 でミニゲームを完成させるスキル習得  
- Widget Blueprint を用いた UI 実装 + 3D 基本要素の理解  
- ゲームロジック、UI、演出の一通りの実装経験をポートフォリオに反映  

---

## 🕹 ゲーム内容

- ランダムに生成されるカードから SET を探す  
- SET が存在しなくなった時点でゲーム終了  
- クリアタイムを記録  
- ローカルランキング機能でスコアを保存  
- 初回プレイ時に簡易チュートリアル表示

---

## 🛠 技術的ポイント

### ■ ゲームロジック
- Blueprint のみで構築  
- SET 判定アルゴリズムを実装  
  - 3枚のカードの属性（形・色・塗り・数）を比較  
  - 「全て同じ」または「全て異なる」かで成立判定

### ■ UI / 演出
- カード選択時の強調表示  
- カード取得・再配置時の軽量アニメーション  
- タイマー・スコア UI の実装  
- 初回プレイ用チュートリアル画面

### ■ アセット生成
- Python スクリプトでカード柄を自動生成  
  → 作業効率化と一貫したデザインを実現

### ■ その他
- ランダムなカード配置・リシャッフル処理  
- ローカルランキング保存・表示機能
